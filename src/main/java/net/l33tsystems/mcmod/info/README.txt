Backup your game files prior to using this mod.  It has not been tested with other mods.  Use at your own risk.

Most things in the mod are configurable.  This includes allowing or disallowing parts of the mod to be used.
Please keep in mind that changes to this can negatively effect worlds you have saved that were opened under the original configuration.

If an issue occurs due to this mod being incompatible with another mod, please let us know.
